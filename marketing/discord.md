# Discord Community Setup Guide

## Server Name: GOD — Omniscient Prediction Markets

## Structure

### Channels:
- **#welcome** — Server intro, rules, link to landing page
- **#announcements** — Product updates, new features, launch news
- **#general** — General discussion about predictions, markets, the product
- **#predictions** — Discuss specific GOD predictions, debate outcomes
- **#accuracy** — Track and discuss GOD's accuracy, share case studies
- **#api-dev** — For Prophet tier users, API discussion, integration help
- **#build-in-public** — Behind-the-scenes updates on development
- **#suggestions** — Feature requests, data source suggestions
- **#off-topic** — Non-GOD discussion

### Roles:
- **Seeker** — Free tier members (default role on join)
- **Oracle** — Paid Oracle tier members
- **Prophet** — Paid Prophet tier members (API access)
- **Early Access** — Members who joined during the beta period
- **Moderator** — Community moderators

## Welcome Message:

Welcome to GOD — Omniscient Prediction Markets! 🔮

GOD is a prediction market platform powered by an autonomous ML engine that watches the world 24/7 and forecasts what happens next — with calibrated, explainable probabilities.

**Get started:**
1. Read the rules in #welcome
2. Check out the landing page: https://vishnu-MVP.github.io/god-landing/
3. Join the early access waitlist
4. Head to #predictions to see what GOD is forecasting
5. Share your thoughts in #general

**Roles:**
- Seeker (Free) — browse markets and predictions
- Oracle ($29/mo) — dark signal alerts, causal chains, watchlists
- Prophet ($99/mo) — full API access for developers

We're building in public. Your feedback shapes the product. Let's build the future of forecasting together. 🚀

## Rules:
1. Be respectful — no harassment, hate speech, or personal attacks
2. No spam or self-promotion (except in designated channels)
3. Keep discussions on-topic per channel
4. Predictions are probabilistic — no guarantee of outcomes, not financial advice
5. Have fun and learn together

## Setup Instructions:
1. Create Discord server with the name above
2. Create all channels listed
3. Set up roles and permissions
4. Post welcome message in #welcome
5. Create invite link and share on landing page, Twitter, Reddit
6. Seed initial content — post a few predictions, accuracy updates, and a build-in-public update
7. Invite 5-10 early supporters to seed the community
